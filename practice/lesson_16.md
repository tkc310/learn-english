# Lesson16

## Lesson15の復習

### 1. 疑問詞を入れる

what / who / which / whose / how / when

1. `You can get there by bus.` -> `[ How ] can i get to the place?`
2. `It's June 28.` -> `[ When ] is your birthday?`
3. `Naoko did.` -> `[ Who ] called you?`
4. `The purple one.` -> `[ Which ] dress is yours?` -> ~one なので which
5. `He is smoking ouside.` -> `[ What ] is he doing?`

### 2. 誤った英文を直す

1. (b)  
`What color did your sister choose?` -> what は color の疑問形容詞なので直後にcolorを配置
2. (e)
`Which train will you take tomorrow?` -> 疑問形容詞 which train の後は疑問文になるため will you ~ が正しい

### 3. 適切な単語を入れる

1. この建物はどれくらい古い？  
`[ How ] [ old ] [ is ] this building?`
2. あの黒い車は誰のですか？  
`[ Whose ] [ is ] that black car?`
3. パンを英語で何という？  
`[ What ] [ do ] you [ call(say) ] "pan" in English?`


## Lesson16の演習

### 1. 適切な前置詞を選ぶ

1. `They go hiking ( in ) spring.` -> 季節 in
2. `We practice kendo ( on ) Saturday.` -> 曜日 on
3. `It rains a lot ( in ) June.` -> 月 in
4. `You go to bed ( at ) 10 o'clock.` -> 時間 at
5. `He opened a new shop ( on ) April 1.` -> 特定日 on

### 2. 適切な前置詞を記入

1. 彼らは月曜日にスペイン語を学ぶ予定だ  
`They will learn Spanish [ from ] Monday.`
2. まず第一に私たちは水曜日までにこの仕事をしなければならない  
`First of all, we must do this job [ by ] Wednesday.` -> first of all で「まず第一に」
3. 私はある日彼の友達の一人に会った  
`I met a friend [ of ] his one day.` -> one day で「ある日」
4. 私は大阪のホテルで働いている  
`I work [ at ] a hotel [ in ] Osaka.`
5. その店の前にバス停がある  
`There is a stop [ in ] [ front ] [ of ] the shop.`
6. 私の父は犬と一緒に公園に行った  
`My father went [ to ] the park [ with ] my dog.`


### 3. 指定の前置詞句が 形容詞A or 副詞B か選ぶ

1. `My favorite flowers (in the garden) were very beautiful.` (A) -> 名詞 flowers を修飾
2. `We were (in Okinawa).` (B) -> be動詞 were を修飾
3. `They are (from south Africa).` (A) 代名詞 They を修飾
4. `We run (in the park).` (B) 動詞 run を修飾
5. `I met him (at the station).` (B) 動詞 met を修飾
