# Lesson18

## Lesson17の復習

### 1. 並び替える

1. like / does / to draw / your father / pictures / ?   
`Does your brother like to draw pictures?`
2. went / one day / to / foreign(外国の) animals / we / to see / the zoo / .   
`We went to the zoo to see foreign animals one day.`
3. an hour / this job / it / finish / takes / to / .  
`It takes an hour to finish this job.`
4. doesn't / use / my brother / how / know / a computer / to / .  
`My brother doesn't know how to use a computer.`
5. weight / her / to / is / goal / lose / .  
`Her goal is to lose weight. (彼女の目標は体重を減らす事だ)`


### 2. 適切な単語を入れてほぼ同じ内容の英文にする

1. It was very important to decide where to stay.  
`[ To ] [ decide ] where to stay [ was ] very important.`
2. I want to drink something.  
`I want [ something ] to [ drink ].`


### 3. 正しい位置に不定詞を入れる

1. The girl loves [ to sing ] the song. -> SVO 名詞的用法
2. You have many things [ to do ]. -> SVO 名詞的用法
3. We went to the cafeteria [ to study ]. -> SV+M 副詞的用法
