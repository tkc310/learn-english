# Lesson18

## Lesson17の復習

### 1. 並び替える

1. like / does / to draw / your father / pictures / ?   
`Does your brother like to draw pictures?`
2. went / one day / to / foreign(外国の) animals / we / to see / the zoo / .   
`We went to the zoo to see foreign animals one day.`
3. an hour / this job / it / finish / takes / to / .  
`It takes an hour to finish this job.`
4. doesn't / use / my brother / how / know / a computer / to / .  
`My brother doesn't know how to use a computer.`
5. weight / her / to / is / goal / lose / .  
`Her goal is to lose weight. (彼女の目標は体重を減らす事だ)`


### 2. 適切な単語を入れてほぼ同じ内容の英文にする

1. It was very important to decide where to stay.  
`[ To ] [ decide ] where to stay [ was ] very important.`
2. I want to drink something.  
`I want [ something ] to [ drink ].`


### 3. 正しい位置に不定詞を入れる

1. The girl loves [ to sing ] the song. -> SVO 名詞的用法
2. You have many things [ to do ]. -> SVO 名詞的用法
3. We went to the cafeteria [ to study ]. -> SV+M 副詞的用法


## Lesson18の演習

### 1. 正しい単語を選択

1. He wanted ( training ) his dog. -> wantの目的語は不定詞
2. Please enjoy ( drinking ) now. -> enjoy/finish/stopの目的語は動名詞
3. Don't give up ( going ) to Tokyo. -> give upの目的語は動名詞
4. Ayako is good at ( playing ) volleyball. -> 前置詞 to の後ろは動名詞
5. I hope ( to get ) some fresh air. -> hopeの目的語は不定詞
6. The artist stopped ( working ) in Paris. -> stopの目的語は動名詞


### 2. 正しい単語を入れる

1. 新しい文化について学ぶのは、とてもワクワクする  
`[ To ] [ learn ] about a new cultures is very exciting.` -> 「To + 動詞の原形」で主語になる
2. メールを私に送ってくれてありがとう  
`Thank you for [ sending ] an email [ to ] me.` -> 「Thank you for -ing」で「〜してくれてありがとう」
3. 僕の仕事は病気の人を助ける事だ  
`My job is [ helping ] sick people.`
4. 彼は毎日ギターを弾く練習をする  
`He practices [ playing ] [ the ] guitar every day.`


### 3. 正しい英文を選ぶ

1. 手紙を送ったことを覚えている？  
`Do you remember sending a latter?` (b) -> remember -ingで「〜したことを覚えている」
2. 彼女は彼の小袋を持ってくることを忘れました  
`She forgot to bring his package.` (a) -> forget toで「〜することを忘れる」
3. 彼が試しにビザを申請してみた  
`He tried applying for a visa.` (a) -> try -ingで「〜することを試してみる」
