# Lesson14

## Lesson13の復習

### 1. 英文を完成させる

1. 彼は７月に新しいスポーツを始めるつもりはない
`He [ isn't ] [ going ] [ to ] [ begin ] a new sport in July.`
2. 明日、雪は降らないだろう  
`[ It ] [ won't ] snow [ tomorrow ].`
3. 来年、彼女は20歳になる  
`She [ will ] [ be ] 20 yeas old [ next ] [ year ].`
4. 今夜、あなたは出かけるつもりですか？
`[ Are ] you [ going ] [ to ] go out [ tonight ]?`

### 2. 単語を並び替える

1. a school / next / to  / will / year / graduate / from / you / ?  
`Will you graduate from a school next year?`
2. this jacket / going / tomorrow / are / will / wear / you / to / ?  
`Are you going to wear this jacket tomorrow?`

### 3. 指定した数の単語の英文にする  

1. 彼は家を買わないだろう 5  
`He won't buy a house.`
2. 私は映画を見るつもりだ 6    
`I'm going to watch[see] the[a] movie.`
