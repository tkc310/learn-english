# 5文型

- S - Subject 主語
- V - Verb 動詞
- O - Object 目的語 (名詞のみ)
- C - Complement 補語 (名詞 or 形容詞)
- M - Modifier 修飾語

|文型|構成|例文|補足|
|--|--|--|--|
|SV[+M]|主語 + 自動詞 [+修飾語]|You fight. / You fight in a war.||
|SVC|主語 + 不完全自動詞 + 主格補語|You are safe.|S=Cの関係 / 不完全動詞は補語が必要とする動詞|
|SVO|主語 + 他動詞 + 目的語|You play the guitar.|S≠Oの関係 / 他動詞は目的語を必要とする動詞|
|SVOO|主語 + 他動詞(授与動詞) + 間接目的語(O1) + 直接目的語(O2)|You tell me a lie. <-> You tell a lie to me.|O1≠O2の関係 / 授与動詞は2つの目的語を必要とする動詞 / SVOO <-> SVO+Mに変換可能|
|SVOC|主語 + 不完全他動詞 + 間接目的語(O1) + 目的格補語|You leave me alone.|O=Cの関係 / 不完全他動詞は目的語と補語を必要とする動詞|

## 前置詞 / 前置詞句について

```
You walk in the park.
```

上記は `S(you) + V(walk) + M(in the park)` というSV文型であるが、
Mは `in` という前置詞が `the park` という名詞の前についた **前置詞句** になっている。

## 場所と時間の修飾語の位置について

修飾語はいくつでも利用して良いが `S + V + M(場所) + M(時間)` の順番で配置しなければならない。

```
I run in the park in the morning.
You come here every morning.
```

## be動詞が存在を意味する場合

be動詞が存在を意味する場合は、SVC文系ではなくSV文型になる。

```
He is a student. -> SVC 彼は生徒だ
He is in the library. -> SV+M 彼は図書室にいる (図書室に〜という修飾語は必須でない)
```

## 形容詞 / 副詞について

下記の違いがある。
- 形容詞 - 名詞のみを修飾
- 副詞 - 主に動詞(名詞以外)を修飾

下記の配置で利用する。

- 形容詞
```
形容詞 -> 副詞
```
- 副詞
```
be動詞 -> 副詞(頻度) -> 一般動詞 -> 目的語 -> 副詞(程度)
```

## to / forについて

- to - 方向や到達点 (〜に向けた、〜宛に)
- for - 目的 (〜のために)

## itは直接目的語(O2)にできない

itは **代名詞** のため `SVO+M` 文型で利用する。

```
I buy it for you. 🙆
I buy you it. 🙅
```
