# 不定詞

## 不定詞の役割

「to + 動詞の原形」で不定詞になる。  
不定詞の理解には「名詞・形容詞・副詞」の知識が必要になる。

- 名詞: 主語(S) / 目的語(O) / 補語(C)
- 形容詞: 補語(C) / 名詞の修飾語(M)
- 副詞: 名詞以外の修飾語(M) -> 形容詞,文全体,動詞,他の副詞など


## 不定詞の3用法

### 名詞的用法 〜すること -> 主語(S) / 目的語(O) / 補語(C)





### 形容詞的用法 〜するための、するべき -> 名詞を修飾





### 副詞的用法 〜するために、〜して -> 目的・原因を説明
