# 接続詞

## 接続詞とは

### カタマリ製造機

接続詞は何かと何かを繋いで「カタマリ」を作る働きをする。  

He has a frog. + He has a snake, too.  
-> He has a frog and a snake.

Tom got angry. + Tom went home. + Kerry came late.  
-> Tom got angry and went home because Kerry came late.


### 句と節

文を構成する区分は句と節に分けられる。  
これらは名詞・形容詞・副詞と同じ役割を果たす。

**句の例文**  
- [ Your father (名詞句=修飾語+名詞)] is [ from Hokkaido (形容詞句=前置詞句)].
- [ To teach you English every day (名詞句=不定詞)] is [ my job (名詞句=修飾語+名詞)].

「修飾語+名詞 / 不定詞 / 前置詞句」のように2語以上の単語で構成されるカタマリを「句」と呼ぶ。  
- 名詞的役割をする句を「名詞句」
- 形容詞的役割をする句を「形容詞句」

**節の例文**  
- I think that [ he uses a dictionary (目的語となる名詞節)].

カタマリにSVが含まれていれば節である。  

語・句・節の関係性は下記のようになる。  
dictionary (語) < useful dictionary (句=2語以上) < he uses a dictionary (節=最低でもSVが含まれる)
