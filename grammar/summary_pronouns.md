## 数量を表す限定詞・不定代名詞

# 数量を表す限定詞・不定代名詞の一覧表

名詞の前に置く「限定詞（形容詞のような働き）」と、単独で名詞の代わりに使う「不定代名詞」の両方で使えるものが多くあります。**「数えられる名詞（可算名詞）」につくか、「数えられない名詞（不可算名詞）」につくか**で使い分けます。

---

### 1. 数量を表す言葉の一覧表（量・数別）

| 数量の度合い | 可算名詞（複数形）<br>（本、人、車など） | 不可算名詞<br>（水、お金、時間など） | 両方に使える語 |
| :--- | :--- | :--- | :--- |
| **すべて / 全体**<br>（100%） | **all / every**（※1） / **each** | **all** / **whole** | **all** |
| **たくさん / 大部分**<br>（70〜90%） | **many / a large number of** / **most** | **much / a large amount of** / **most** | **a lot of / lots of / plenty of** / **most** |
| **いくつか / いくらか**<br>（30〜50%） | **some**（肯定） / **any**（否定・疑問） | **some**（肯定） / **any**（否定・疑問） | **some / any** |
| **少しある**<br>（肯定的：10〜20%） | **a few** / **several**（数個の） | **a little** | - |
| **ほとんどない**<br>（否定的：0〜5%） | **few** | **little** | - |
| **ゼロ / 全くない**<br>（0%） | **no** | **no** | **no / none**（※2） |

* （※1）**every** は「すべての」という意味ですが、後ろには必ず**単数名詞**が来ます（例：*every day*）。
* （※2）**none** は「代名詞」としてのみ使います。後ろに直接名詞を置くことはできません（例：*None of the food was left.* / *None of them came.*）。

---

### 2. 「2つのもの」に対して使う数量限定詞・不定代名詞

対象が「2つ（2人）」と決まっている場合は、専用の単語を使います。

| 意味 | 単語 | 使い方・例文 |
| :--- | :--- | :--- |
| **両方とも** | **both** | 後ろは複数名詞。<br>例：*Both eyes*（両目）/ *Both of them*（彼らの両方） |
| **どちらか一方** | **either** | 後ろは単数名詞。<br>例：*either side*（どちら側でも） |
| **どちらも〜ない** | **neither** | 後ろは単数名詞（否定文にする）。<br>例：*Neither plan is perfect.*（どちらの計画も完璧ではない） |

---

### 💡 覚えるための重要ポイント

* **a が「ある」かないかの違い**：
  * **a** few / **a** little ＝「**少しある**」（肯定的）
  * few / little ＝「**ほとんどない**」（否定的）
  「a」という文字が1つ「ある」から「少しある」と覚えると忘れません。
* **限定詞と代名詞の形の変化**：
  多くの語は「*some* books（限定詞）」としても「I want *some*.（代名詞）」としても同じ形で使えます。しかし、**no** だけは代名詞になると **none** に形が変わるので注意しましょう（❌ *I have no.* ➡️ ⭕️ *I have none.*）。
